# "빠르게 배우고 새로운 걸 배우는 것을 즐기는 개발자"
새로운 기술과 지식을 빠르게 익히며, 도전에 대한 호기심과 열정으로 꾸준히 성장하는 개발자입니다. 특히 백엔드 개발에 큰 흥미와 열정을 가지고 있으며, 코딩을 통해 지식을 확장하고 기술을 습득하는 과정을 즐깁니다. <br>
빠른 학습 능력을 바탕으로, 더 나은 백엔드 개발자가 되기 위해 지속적으로 노력하겠습니다.
